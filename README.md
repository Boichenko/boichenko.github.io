# Hanna Boychenko React portfolio

React portfolio project generated from the provided PDF resume.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Main content lives in `src/App.jsx`
- Styling lives in `src/styles.css`
- Profile image extracted from the PDF is in `src/assets/profile.jpg`
- The LinkedIn URL was not available as a full link in the parsed PDF text, so the placeholder text in the contact card should be replaced manually if needed
